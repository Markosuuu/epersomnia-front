# Epersomnia

Frontend para el desarrollo del último trabajo práctica de _Estrategias de persistencia_.

## Manera de levantarlo:
1. Clonar el repo.
2. abrir la consola y escribir:
   1. `pnpm i` <--- esto instala las dependencias
   2. `pnpm dev` <--- esto corre la aplicación en local


**Aclaración**: Tener _pnpm_ instalado.

## Tecnologías
* React
* Typescript
* Tailwind
